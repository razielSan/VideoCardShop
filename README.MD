## 🚀 Quick Start

### 1. Install poetry globally on your computer

```bash
pip install poetry
```

### 2. Clone repository


```bash
git clone git@github.com:razielSan/VideoCardShop.git
cd VideoCardShop
```


### 3. Install dependencies


```python
poetry install
```

### ▶️ Run Application

```bash
poetry run uvicorn app.main:app --reload
```
or

```bash
poetry run app
```
